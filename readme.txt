1.let and const command
	a.let command(example:let.js)
		let用于声明变量，用法与var相似，但是所声明的变量只在let命令所在的代码块内有效;
		let应用场景：for循环的计数器;
		不存在变量提升;
		暂时性死区(temporal dead zone,简称TDZ);
		不允许重复声明:let不允许在相同作用域内重复声明同一个变量;
		块级作用域;
	b.const command
		const用来声明常量,一旦声明,其值就不能改变；
